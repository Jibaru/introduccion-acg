Modelos-3d

La reconstrucción de modelos sólidos 3D a partir de vistas ortográficas 2D, es un algoritmo eficiente para la reconstrucción del modelo tridimensional de cuerpos poliédricos a partir de sus proyecciones ortográficas de planta, alzado y perfil dadas mediante líneas continuas o discontinuas. Dicho algoritmo reduce eficazmente el posible número de objetos ambiguos y puede reconocer todos los sólidos correctos correspondientes a las proyecciones dadas.
